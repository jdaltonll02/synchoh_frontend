# SyncOH Prototype - React (Connected to local backend)

## Quick start

1. Copy `.env` from `.env.example` and set:
   ```
   REACT_APP_API_URL=http://localhost:5000/api
   ```
2. Install:
   ```
   npm install
   ```
3. Start:
   ```
   npm start
   ```

Make sure your backend is running on http://localhost:5000 and seeded.
